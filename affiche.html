Difficultés Surmontées

  • Sécurisation LLM : Détection robuste des attaques "Prompt Injection" (ignore previous instructions) via Regex compilées et clauses système.
  • Indexation Audio : Transcription Whisper CUDA → ChromaDB en chunks 60s, avec metadata vidéo (YOLOv1 from Scratch...).
  • Routeur Multi-Catégories : Flux LangChain asynchrone forçant JSON strict pour filtrage sémantique (Legal/Research/Technical/Audio).

Prochaines Étapes (En cours)

  • Multi-LLM : Intégration agents (Groq + Ollama local) avec fallback et évaluation automatique.
  • ML Feedback : Fine-tuning embeddings sur logs feedback pour personnalisation utilisateur.
